# Security Policy

## Supported Versions

The main branch receives security updates. Released versions follow semantic versioning.

## Responsible Use

Phantomwire is for authorised security assessment and research only. Always obtain explicit
permission before targeting systems. Enable safe-mode consent by setting:

```bash
export PHANTOMWIRE_CONSENT=I_HAVE_PERMISSION
```

Limit allowed scopes to hosts you control via CLI flags or configuration.

## Reporting a Vulnerability

Please report suspected vulnerabilities to maintainers@phantomwire.org. Include:

- Description of the issue and potential impact
- Reproduction steps or proof-of-concept (when safe)
- Suggested mitigations

We will acknowledge receipt within five business days and provide updates until resolution.
Do not test against production systems without authorization.
